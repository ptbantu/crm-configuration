# BANTU CRM 文档目录

## 文档概览

本目录包含 BANTU CRM 系统的设计文档、业务说明和技术文档。

### 1. 系统架构
- **`module_architecture.md`** - 系统功能模块架构划分

  - 模块开发优先级
  - 模块间依赖关系
  - 技术实现建议

### 2. 业务流程

  - 销售创建订单流程

### 3. 客户管理

## 文档使用说明

### 开发参考
- 开发新功能时，参考 `module_architecture.md` 了解模块划分和功能点
- 实现业务流程时，参考 `visa_service_workflow_implementation.md` 了解数据流和 SQL 示例

### 业务理解
- 理解客户分类逻辑，参考客户分类相关文档
- 了解系统整体架构，参考模块架构文档

## 文档维护

- 文档应与代码和数据库架构保持同步
- 重大架构变更时，及时更新相关文档
- 废弃的功能和概念，及时删除相关文档
